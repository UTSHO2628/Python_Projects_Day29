# University_Campus_Voice_Bot  
# Database {XXAMP} using for question & answer memory
 
A Python-based Campus Voice Assistant that allows users to interact with a campus information system using voice commands. The application uses speech recognition to understand user queries, fetches relevant answers from a MySQL database, responds via text-to-speech, and enhances user interaction with a Tkinter GUI and animated video feedback.
